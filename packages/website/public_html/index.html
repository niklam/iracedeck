Three Plugins, One Suite

Modular Design

Pit Service

Pit Plugin

Complete pit stop control at your fingertips. Add or reduce fuel with precision, change tires, switch compounds between dry and wet, and request fast repairs.

  • Fuel adjustment (+/- with configurable amounts)
  • Tire change with individual wheel control
  • Dry/Wet compound switching
  • Fast repair request
  • Live fuel-to-add display
Communication

Comms Plugin

Quick chat macros for rapid race communication. Send pre-configured messages with a single tap — apologize, congratulate, or coordinate with your team.

  • Customizable chat messages
  • One-tap message sending
  • "Sorry!", "GG!", "GL HF" and more
  • Team radio integration
  • No typing required mid-race
Core

Main Plugin

Essential telemetry displays and vehicle information. See your speed, gear, weather conditions, and more — all updating in real-time on your Stream Deck.

  • Speed display (km/h or mph)
  • Current gear indicator
  • Weather/sky conditions
  • Real-time telemetry updates
  • Direct SDK integration
