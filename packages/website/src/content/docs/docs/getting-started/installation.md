## Requirements

- **Stream Deck software** version 7.1 or newer
- Any Stream Deck device: Stream Deck, Stream Deck Mini, Stream Deck XL, or Stream Deck+

## Option 1: Install from Elgato Marketplace

The easiest way to install iRaceDeck:

1. Open the [Elgato Marketplace listing](https://marketplace.elgato.com/product/iracedeck-042a0efb-58aa-428c-b1de-8b6169edd21d)
2. Click **Install** — the Stream Deck software will handle the rest
3. Once installed, look for the **iRaceDeck** category in the Stream Deck action list

## Option 2: Install from GitHub Releases

GitHub Releases may have a newer version available before it appears on the Elgato Marketplace. Use this option if you want the latest version or if the Marketplace install doesn't work:

1. Go to the [GitHub Releases page](https://github.com/niklam/iracedeck/releases/)
2. Find the latest release and download the file named **`iracedeck-vX.X.X.streamDeckPlugin`** (e.g., `iracedeck-v1.0.0.streamDeckPlugin`)
3. Double-click the downloaded `.streamDeckPlugin` file
4. The Stream Deck software will open and install the plugin automatically
5. Look for the **iRaceDeck** category in the Stream Deck action list

## After Installation

Drag any iRaceDeck action onto your Stream Deck profile. Most actions have a **mode** dropdown in the Property Inspector (the settings panel on the right) that lets you choose what the button does.

For actions that use keyboard shortcuts (like black box selection or camera controls), you can customize the key bindings in each action's Property Inspector to match your iRacing configuration. See [Key Bindings](/docs/features/key-bindings/) for more details.

## Stream Deck+ Encoder Support

Most iRaceDeck actions support the Stream Deck+ encoder (rotary dial). When you place an action on an encoder slot, rotating the dial typically cycles through options or adjusts values, while pressing the dial activates the primary action.
