Complete reference of iRacing keyboard shortcuts, their defaults, and whether they are available via the iRacing SDK.

Actions marked "Available via SDK" use SDK commands directly and don't require keyboard shortcut configuration. Actions without SDK support use configurable keyboard shortcuts that can be customized in each action's Property Inspector.
