The Splits Delta Cycle action lets you switch between iRacing's different splits delta display modes. Quickly find the delta comparison that's most useful for your current session.

## Modes

| Mode | Description |
|------|-------------|
| Next Delta Mode | Cycles to the next splits delta display mode. |
| Previous Delta Mode | Cycles to the previous splits delta display mode. |

## Encoder Support

Yes — rotation cycles through delta modes (clockwise for next, counter-clockwise for previous).
