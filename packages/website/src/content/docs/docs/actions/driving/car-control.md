The Car Control action provides quick access to essential car functions. Toggle the starter, ignition, pit speed limiter, or request a tow — all from a single button.

## Modes

| Mode | Description |
|------|-------------|
| Starter | Engages the car starter. |
| Ignition | Toggles the ignition on or off. |
| Pit Speed Limiter | Toggles the pit speed limiter. Icon updates based on telemetry to reflect current state. |
| Enter/Exit/Tow | Enters or exits the car, or requests a tow to pit lane. |
| Pause Sim | Pauses the simulation. |

## Encoder Support

Yes.

:::note
The Pit Speed Limiter mode features a telemetry-aware icon that reflects the current limiter state in real time.
:::
