Tire Service handles everything tire-related for your pit stops. Change all tires at once, toggle individual wheels, clear tire selections, or switch between dry and wet compounds.

:::note
Tire Service is telemetry-aware — the compound display updates with live data from the sim so you always see the current tire state.
:::

## Modes

| Mode | Description |
|------|-------------|
| Change All Tires | Request new tires on all four corners (uses iRacing `#t` pit chat macro) |
| Clear Tires | Clear all tire change requests |
| Toggle Tires | Toggle tire change per wheel (LF, RF, LR, RR) |
| Change Compound | Cycle through available tire compounds (DRY/WET) |

## Encoder Support

Yes — rotation cycles through tire options or adjusts the selected setting with the Stream Deck+ dial.
