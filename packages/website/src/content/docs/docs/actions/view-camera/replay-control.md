Replay Control gives you complete command over iRacing's replay system. Playback transport, speed adjustments, and session/lap/incident navigation are all available as selectable modes on a single action.

:::note
Replay Control replaces the legacy Replay Transport, Replay Speed, and Replay Navigation actions. Existing button configurations using those actions will continue to work.
:::

## Modes

| Mode | Description |
|------|-------------|
| Play/Pause | Toggle replay playback |
| Stop | Stop replay playback |
| Fast Forward | Fast-forward through the replay |
| Rewind | Rewind through the replay |
| Slow Motion | Play the replay in slow motion |
| Frame Forward | Advance one frame |
| Frame Backward | Step back one frame |
| Increase Speed | Increase replay playback speed |
| Decrease Speed | Decrease replay playback speed |
| Next Session | Jump to the next session |
| Previous Session | Jump to the previous session |
| Next Lap | Jump to the next lap |
| Previous Lap | Jump to the previous lap |
| Next Incident | Jump to the next incident |
| Previous Incident | Jump to the previous incident |
| Jump to Beginning | Jump to the start of the replay |
| Jump to Live | Jump to the live session |

## Encoder Support

Yes — rotation steps through speed or position depending on the selected mode.
