# Toggle Display Reference Car

> **Note:** This functionality has moved to the [Splits & Reference](splits-delta-cycle.md) action as the "Toggle Reference Car" mode. The standalone toggle-display-reference-car action is no longer available. The "Display Reference Car" option in the Toggle UI Elements action is deprecated but still works for backward compatibility.

Toggles the reference car display for split-time comparison.

## Properties

| Property | Value |
|----------|-------|
| Action ID | `com.iracedeck.sd.core.splits-delta-cycle` (toggle-ref-car mode) |
| Type | Multi-toggle |
| SDK Support | No |
| Encoder Support | Yes |

## Migration

To use this functionality:
1. Add a **Splits & Reference** action to your Stream Deck
2. Set Mode to **Toggle Reference Car** in the Property Inspector

## Keyboard Simulation

| Action | Default Key | iRacing Setting |
|--------|-------------|-----------------|
| Toggle | Ctrl+C | Toggle Display Reference Car |

## Notes

- Shows which car you're comparing split times against
- Useful for tracking specific competitors during a race
